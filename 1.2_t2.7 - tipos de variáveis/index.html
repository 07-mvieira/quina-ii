médias tipos de variáveis matemática tarefas sorteio

var

variável com escopo global, que se aplica para todo o código JavaScript sem exceção. até 2015, era o único tipo de variável.


let

variável com escopo de bloco, se aplicando apenas para o bloco de código em que ela reside. preferível na maioria dos casos.


const

constante com escopo de bloco, que, diferente de uma variável, não pode ser redefinida após sua declaração inicial. útil para, por exemplo, manipular um elemento HTML.
